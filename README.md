# Titanic Data Analysis (EDA Project)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to analyze passenger survival patterns based on features like gender, age, and passenger class.

## Dataset
- Source: https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv
- Contains details such as:
  - Age
  - Gender
  - Passenger Class
  - Fare
  - Survival status

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Google Colab / Jupyter Notebook
  
## Tasks Performed
- Data loading and understanding
- Handling missing values
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization

## Key Insights
- Female passengers had a higher survival rate than males  
- 1st class passengers had better survival chances  
- Age had an impact on survival probability  

## Visualizations
- Countplot (Survival by Gender & Class)
- Histogram (Age distribution)
- Boxplot (Age vs Survival)
- Pairplot (Feature relationships)
